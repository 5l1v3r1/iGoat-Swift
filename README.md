# OWASP iGoat (Swift) - A Damn Vulnerable Swift Application for iOS #

This is a Swift version of original iGoat https://github.com/OWASP/igoat project. 

Using OWASP iGoat, you can learn exploiting and defending vulnerabilities in iOS Swift applications. 

### Vulnerabilities Covered (version 1.0): ###
* __OWASP TOP 10 Mobile__
* __Reverse Engineering__
* __Data Protection (Rest)__
* __Data Protection (Transit)__
* __Key Management__
* __Tampering__ 
* __Injection Flaws__
* __Broken Cryptography__
* __URL Scheme Attack__
* __Social Engineering__
* __Authentication__
* __Side Channel Data Leaks__
